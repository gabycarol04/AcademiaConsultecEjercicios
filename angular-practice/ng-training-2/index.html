Apuntes

  • Modulo Padre
    var app = angular.module('AppModule', []);
  • Para hacer el binding de las variables (entre controlador y template) hay dos formas: usando $scope o usando un alias
  • Controlador con scope
    app.controller('UserController', function($scope)
    //Parte 1: con Scope
    $scope.aula = "Programadores";
    });
  • Controlador con alias
    app.controller('AliasController', function(){
    //Parte 2: con alias
    this.nombre = "Gabriela Navarro";
    });
  • Se puede tener un controlador dentro de otro
